# 🌸 건강 다이어리

**달력을 중심으로** 몸무게 · 식사 · 월경 주기를 한 곳에서 기록하는 웹 앱입니다.
설치나 서버 없이 **`index.html` 파일을 브라우저에서 열기만** 하면 바로 사용할 수 있고,
모든 데이터는 사용 중인 브라우저(localStorage)에만 저장됩니다.

앱의 기본 화면은 **달력**입니다. 각 날짜 칸에 그 날의 몸무게 · 식사 체크 · 월경 상태가
함께 표시되고, **날짜를 누르면** 그 날의 몸무게 · 식사 · 월경을 한 번에 입력/수정할 수 있습니다.

## 설치 & 실행

### 🔧 빠른 시작 (로컬 파일)
`index.html` 파일을 브라우저로 열면 바로 실행됩니다.

### 📱 PWA 설치 (권장)
더 나은 환경을 위해 **Progressive Web App으로 설치**할 수 있습니다:

**Android / Chrome:**
1. 앱 열기 → 브라우저 메뉴(⋯) → "설치" 또는 "홈 화면에 추가"
2. 홈 화면에 앱 아이콘 생성 → 아이콘 탭해서 실행

**iPhone / Safari:**
1. 주소창의 공유 버튼 → "홈 화면에 추가"
2. 홈 화면에 앱 아이콘 생성 → 아이콘 탭해서 실행

**데스크톱 (Chrome/Edge):**
1. 주소창 우측의 설치 아이콘(⬇️) 클릭
2. 또는 F12 DevTools → Application → Manifest에서 확인

설치하면:
- 📱 앱처럼 전체 화면에서 실행
- 📡 오프라인에서도 작동
- ⚡ 빠른 로딩 (캐싱)

### 🖥️ 로컬 서버 실행
HTTPS가 필요한 기능을 테스트하려면:

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
# (로컬 http에서는 일부 PWA 기능 제한)
```

## 화면 구성

### 📅 달력 (기본 화면)
- 한 달 달력에 모든 기록이 모입니다.
- 각 날짜 칸 표시:
  - 배경색: 월경 / 예상 월경 / 가임기 / 배란일
  - 🩸 / ✦ : 월경일 · 배란일 표시
  - ❤ : 부부관계 기록이 있는 날
  - 숫자: 그 날의 몸무게(kg)
  - 점 3개: 아침·점심·저녁 식사 여부(초록=먹음)
- **날짜를 누르면** 상세 입력 패널이 열려 한 번에 기록:
  - ⚖️ 몸무게 입력 / 지우기
  - 🍽️ 아침·점심·저녁 체크 + 무엇을 먹었는지 메모
  - 🩸 이 날을 월경 시작일로 기록 / 해제
  - ❤️ 부부관계 여부 기록 (가임기·배란일이면 임신 가능성 안내)
- **주기 설정**: 평균 주기(기본 28일) · 월경 기간(기본 5일), 시작일 2회 이상 기록 시 **평균 주기 자동 계산**
- **예측 요약** (알림 토글 포함):
  - **다음 월경 예정일** = 마지막 시작일 + 주기
  - **배란일** = 다음 생리 예정일 − 14일
  - **가임기** = 배란일 −5일 ~ 배란일 +3일
  - **🔔 알림** — 토글을 켜면 다음 월경 예정일 전날(D-1)과 당일에 브라우저 알림으로 미리 알려줍니다 (권한 필요)

### 📊 추이 · 기록
- 몸무게 변화 추이 선 그래프 + 시작 대비/일별 증감 통계
- 몸무게 기록 목록
- 최근 14일 식사 기록

## 💾 백업 · 복원
‘추이 · 기록’ 탭의 **백업 · 복원** 카드에서:
- **백업 파일 내보내기** — 모든 기록을 JSON 파일로 저장 (파일명: `건강다이어리-백업-YYYY-MM-DD.json`)
- **백업에서 복원** — 백업 파일을 선택해 복원. 복원 전 포함 내용을 보여주고 **확인**을 거쳐 현재 데이터를 덮어씁니다.
- 현재 보관 중인 기록 수(몸무게/식사/월경/부부관계)와 **최근 백업 시각**을 표시합니다.
- 잘못된 파일은 검증 후 거부하며, 기존 데이터는 보존됩니다.

브라우저 데이터를 지우면 기록이 사라지므로, 기기 변경 전이나 주기적으로 백업을 권장합니다.

## 🧪 자체 테스트
실제 브라우저(Playwright)로 주요 기능을 검증하는 테스트가 포함되어 있습니다.

```bash
node selftest.mjs   # 몸무게·식사·부부관계·월경 예측·백업/복원 등 검증
```
