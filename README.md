# 🌸 건강 다이어리

몸무게 · 식사 · 월경 주기를 한 곳에서 기록하는 간단한 웹 앱입니다.
설치나 서버 없이 **`index.html` 파일을 브라우저에서 열기만** 하면 바로 사용할 수 있고,
모든 데이터는 사용 중인 브라우저(localStorage)에만 저장됩니다.

## 사용 방법

```
index.html 파일을 더블클릭하거나 브라우저로 열면 끝!
```

또는 로컬 서버로 열고 싶다면:

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```

## 기능

### ⚖️ 몸무게
- 날짜별 몸무게 입력
- 변화 추이 그래프 (선 그래프)
- 시작 대비 증감, 일별 변화량 표시
- 기록 목록 및 개별 삭제

### 🍽️ 식사
- 날짜별 아침 / 점심 / 저녁 먹었는지 체크
- 각 끼니마다 무엇을 먹었는지 메모
- 최근 14일 식사 기록 한눈에 보기
- 변경 시 자동 저장

### 🩸 월경 달력
- 달력에서 날짜를 눌러 월경 시작일 기록/해제
- 평균 주기(기본 28일) · 월경 기간(기본 5일) 설정
- 시작일을 2회 이상 기록하면 **평균 주기 자동 계산**
- 예측 표시:
  - **다음 월경 예정일** = 마지막 시작일 + 주기
  - **배란일** = 다음 생리 예정일 − 14일
  - **가임기** = 배란일 −5일 ~ 배란일 +3일
- 색상 구분: 월경 / 예상 월경 / 가임기 / 배란일 / 오늘

## 데이터 백업
하단의 **데이터 내보내기 / 가져오기** 버튼으로 JSON 파일 백업·복원이 가능합니다.
(브라우저 데이터를 지우면 기록이 사라지므로 가끔 내보내기를 권장합니다.)

## 파일 구조
```
index.html   화면 구조
styles.css   스타일
app.js       기능 로직 (저장·차트·달력·예측)
```
