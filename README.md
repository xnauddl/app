# 🌸 건강 다이어리

**달력을 중심으로** 몸무게 · 식사 · 월경 주기를 한 곳에서 기록하는 웹 앱입니다.
설치나 서버 없이 **`index.html` 파일을 브라우저에서 열기만** 하면 바로 사용할 수 있고,
모든 데이터는 사용 중인 브라우저(localStorage)에만 저장됩니다.

앱의 기본 화면은 **달력**입니다. 각 날짜 칸에 그 날의 몸무게 · 식사 체크 · 월경 상태가
함께 표시되고, **날짜를 누르면** 그 날의 몸무게 · 식사 · 월경을 한 번에 입력/수정할 수 있습니다.

## 사용 방법

```
index.html 파일을 더블클릭하거나 브라우저로 열면 끝!
```

또는 로컬 서버로 열고 싶다면:

```bash
python3 -m http.server 8000
# http://localhost:8000 접속
```

## 화면 구성

### 📅 달력 (기본 화면)
- 한 달 달력에 모든 기록이 모입니다.
- 각 날짜 칸 표시:
  - 배경색: 월경 / 예상 월경 / 가임기 / 배란일
  - 🩸 / ✦ : 월경일 · 배란일 표시
  - ❤ : 부부관계 기록이 있는 날
  - 숫자: 그 날의 몸무게(kg)
  - 점 3개: 아침·점심·저녁 식사 여부(초록=먹음)
- **날짜를 누르면** 상세 입력 패널이 열려 한 번에 기록:
  - ⚖️ 몸무게 입력 / 지우기
  - 🍽️ 아침·점심·저녁 체크 + 무엇을 먹었는지 메모
  - 🩸 이 날을 월경 시작일로 기록 / 해제
  - ❤️ 부부관계 여부 기록 (가임기·배란일이면 임신 가능성 안내)
- **주기 설정**: 평균 주기(기본 28일) · 월경 기간(기본 5일), 시작일 2회 이상 기록 시 **평균 주기 자동 계산**
- **예측 요약**:
  - **다음 월경 예정일** = 마지막 시작일 + 주기
  - **배란일** = 다음 생리 예정일 − 14일
  - **가임기** = 배란일 −5일 ~ 배란일 +3일

### 📊 추이 · 기록
- 몸무게 변화 추이 선 그래프 + 시작 대비/일별 증감 통계
- 몸무게 기록 목록
- 최근 14일 식사 기록

## 데이터 백업
하단의 **데이터 내보내기 / 가져오기** 버튼으로 JSON 파일 백업·복원이 가능합니다.
(브라우저 데이터를 지우면 기록이 사라지므로 가끔 내보내기를 권장합니다.)

## 파일 구조
```
index.html   화면 구조 (달력 + 날짜 상세 모달)
styles.css   스타일
app.js       기능 로직 (달력·날짜별 입력·차트·예측)
```
